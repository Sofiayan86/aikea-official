Frequently Asked Questions

What Raspberry Pi model is included in the Full Kit?

The AIKEA Full Kit includes a compatible Raspberry Pi model that's pre-configured with all necessary software for immediate use.

What Raspberry Pi models are compatible with the Light Kit?

The AIKEA Light Kit is compatible with Raspberry Pi 3B, 3B+, and 4. Detailed compatibility information is provided in the installation guide.

Do I need to pay any subscription fees?

No, AIKEA does not require any subscription fees. Once you purchase the device, there are no additional costs.

How difficult is it to set up the Light Kit?

Setting up the Light Kit requires basic familiarity with Raspberry Pi. The installation guide provides step-by-step instructions, but you'll need to install the software yourself.

Can I customize my AIKEA?

Yes! AIKEA is 100% open-source, allowing you to customize both its hardware and software. Many users create custom cases or add additional features.

Is there a warranty?

Yes, all AIKEA products come with a 1-year limited warranty covering manufacturing defects.
